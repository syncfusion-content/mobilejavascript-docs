# Show border

The **data-ej-showborder** attribute is used to specify whether Textbox border is visible or hidden. 

{% highlight html %}


<input type="number" id="textbox_sample" data-role="ejmnumeric" **data-ej-showborder="false"** />


{% endhighlight %}
