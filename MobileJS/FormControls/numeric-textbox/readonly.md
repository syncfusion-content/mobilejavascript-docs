# ReadOnly

The **data-ej-readonly** attribute is used to enable or disable value changes in the Textbox. 

{% highlight html %}


<input type="number" id="textbox_sample" data-role="ejmnumeric"  **data-ej-readonly="false**"/>     


{% endhighlight %}
