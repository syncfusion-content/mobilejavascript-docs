# ReadOnly Support

The ReadOnly property is used to enable or disable the value changes in the Textbox. 

{% highlight html %}

<input type="number" id="textbox_sample" data-role="ejmnumeric"  data-ej-readOnly="false"/>  

{% endhighlight %}
