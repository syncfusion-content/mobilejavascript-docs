# Styles and Appearance

Styles and Appearance of Tab control work based on CSS class. You need a style sheet to apply styles to Tab control that is, ej.mobile.all.min.css.

## Customize styles

In some scenarios, you may want to change color or design of Accordion controlâ€™s appearance. Then, you can override default styles of Accordion control by using its class names. The following are specific class names you can use to modify styles and appearance in Accordion.
